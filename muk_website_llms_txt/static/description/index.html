Overview

Make your Odoo website AI-ready by implementing the llms.txt standard and Cloudflare-style markdown content negotiation. AI agents and crawlers can discover your content via /llms.txt and request any page as clean markdown via the HTTP Accept: text/markdown header. The module automatically indexes all published pages, blog posts, products, and events — no manual configuration needed.

AI-Ready Endpoints

Generates a /llms.txt index following the llmstxt.org standard and a /llms-full.txt with the complete markdown content of every published page — ideal for RAG pipelines and knowledge bases. Any page also responds with clean markdown when an AI agent sends Accept: text/markdown, just like Cloudflare's "Markdown for Agents". Response headers include token counts and your AI usage policy.

Full Control, Zero Config

Works out of the box — pages, blog posts, products, and events are detected automatically. Choose which content types to include and set a Content-Signal policy (train, search, input, or none) to control how AI systems may use your content. Configure everything from Website → Settings under "AI & Agents".
